## Key Topics

- Phase 1: Site excavation and gabion foundation (2 weeks)
- Phase 2: Wall frame and roof rafters (2 weeks)
- Phase 3: Wattle weave and straw-clay insulation (1.5 weeks)
- Phase 4: EPDM membrane and exterior render (1 week)
- Phase 5: Living roof and solar installation (1 week)
- Phase 6: Openings and interior finish (1 week)
- Tool list: excavator, wire cutters, chainsaw, tampers, scaffolding
- Critical path: excavation → gabions → frame → roof → shell
