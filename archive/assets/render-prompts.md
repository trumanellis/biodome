# BioDome Build Guide - 3D Render Specifications

## Technical Requirements
- **Resolution**: 2400 x 1600px (3:2 ratio)
- **Format**: PNG with transparency or clean white/cream background
- **Style**: Clean architectural illustration, warm earth tones
- **Perspective**: Isometric or cutaway section views

## Color Palette
- Eucalyptus timber: #8B7355 (warm brown)
- Cob/earth: #C9A227 (golden earth)
- Stone/granite: #7A7A7A (neutral gray)
- Wire mesh: #A8A8A8 (light metal)
- Living roof greenery: #4A6E4A (forest green)
- Background: #FAF8F5 (cream) or transparent

## Usage Instructions

These prompts are optimized for:
- **DALL-E 3** (OpenAI) - Use prompts as-is
- **Midjourney** - Add `--ar 3:2 --style raw` for architectural precision
- **Stable Diffusion** - Add architectural/technical illustration to negative prompt exclusions

### Recommended Generation Settings
- Aspect ratio: 3:2 (landscape)
- Style: Precise, architectural, technical illustration
- Avoid: Photorealistic, busy backgrounds, excessive detail
